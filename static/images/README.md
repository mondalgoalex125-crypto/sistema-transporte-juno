# Imágenes del Sistema JUNO EXPRESS

## Archivos de imagen necesarios:

1. **bus-hero.jpg** - Imagen principal del bus para la sección hero
2. **chincha.jpg** - Imagen de Chincha (arquitectura colonial)
3. **pisco.jpg** - Imagen de Pisco (playa y costa)
4. **ica.jpg** - Imagen de Ica (oasis y dunas)

## Instrucciones:
Para usar las imágenes reales, guarda los archivos con estos nombres exactos en esta carpeta.
El sistema automáticamente las detectará y las usará en lugar de los placeholders.

## Formatos soportados:
- JPG/JPEG (recomendado)
- PNG
- WebP

## Tamaños recomendados:
- bus-hero.jpg: 1920x1080px (landscape)
- chincha.jpg: 800x600px
- pisco.jpg: 800x600px  
- ica.jpg: 800x600px
